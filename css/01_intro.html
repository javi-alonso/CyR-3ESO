1. Introducción y Sintaxis

La Magia del Diseño

CSS (Hojas de Estilo en Cascada) sirve para cambiar la apariencia de tu HTML. Sin CSS, la web sería muy aburrida (solo texto negro sobre fondo blanco).

Sintaxis Básica

La regla de oro de CSS es:

selector {
  propiedad: valor;
}
  • Selector: ¿A QUIÉN cambio? (Ej: `p`, `h1`, `.miclase`).
  • Propiedad: ¿QUÉ cambio? (Ej: `color`, `font-size`).
  • Valor: ¿CÓMO lo cambio? (Ej: `red`, `20px`).

Ejemplo 1: Texto azul

Selectores de Etiqueta 
<style>
  p {
    color: blue;
    font-size: 20px;
  }
</style>

<p>Soy un párrafo azul.</p>
<p>Yo también, porque soy una etiqueta p.</p>

Soy un párrafo azul.

Yo también, porque soy una etiqueta p.

Ejemplo 2: Usando Clases

Para no cambiar TODOS los párrafos, usamos clases (que empiezan con punto . en CSS).

Selectores de Clase 
<style>
  .alerta {
    color: red;
    font-weight: bold;
  }
</style>

<p class="alerta">¡Cuidado!</p>
<p>Yo soy normal.</p>

¡Cuidado!

Yo soy normal.
