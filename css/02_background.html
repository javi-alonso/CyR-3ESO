2. Fondos (Background)

Más allá del blanco

Podemos cambiar el fondo de cualquier elemento, no solo de la página entera.

1. Color de Fondo

Usamos background-color con nombres de colores (red, blue) o códigos Hexadecimales (#ff0000).

Colores Planos 
<style>
  .caja-verde {
    background-color: #4CAF50; /* Verde */
    color: white;
    padding: 20px;
  }
</style>

<div class="caja-verde">Soy una caja con fondo verde</div>
Soy una caja con fondo verde

2. Imagen de Fondo

Usamos background-image: url('ruta');. Por defecto la imagen se repite si es pequeña.

Imagen de Fondo 
<style>
  .fondo-img {
    background-image: url('https://via.placeholder.com/100');
    height: 150px;
    border: 1px solid black;
  }
</style>

<div class="fondo-img"></div>

3. Ajustando la Imagen

Para evitar que se repita y cubrir todo el espacio.

Fondo Profesional 
<style>
  .fondo-pro {
    background-image: url('https://via.placeholder.com/500x300');
    background-repeat: no-repeat; /* No repetir */
    background-size: cover;       /* Cubrir todo el área */
    background-position: center;  /* Centrar */
    height: 200px;
    color: white;
  }
</style>

<div class="fondo-pro">
  <h2>Texto sobre imagen perfecta</h2>
</div>

Texto sobre imagen perfecta
