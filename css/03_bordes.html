3. Bordes

Definiendo límites

La propiedad border es un atajo para definir grosor, estilo y color en una sola línea.

Sintaxis: Grosor Estilo Color

Bordes Básicos 
<style>
  .solido { border: 2px solid black; }
  .punteado { border: 4px dotted blue; }
  .doble { border: 5px double green; }
</style>

<div class="solido">Borde Sólido</div>
<br>
<div class="punteado">Borde Punteado</div>
<br>
<div class="doble">Borde Doble</div>

Bordes Redondeados (Border Radius)

Para modernizar el diseño, suavizamos las esquinas con border-radius.

Esquinas Redondas 
<style>
  .boton {
    background-color: #2f81f7;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 20px; /* Redondeado suave */
  }
  .circulo {
    width: 50px;
    height: 50px;
    background-color: red;
    border-radius: 50%; /* Círculo perfecto */
  }
</style>

<button class="boton">Botón Redondo</button>
<br><br>
<div class="circulo"></div>

Bordes Específicos

También puedes poner borde solo en un lado: border-bottom, border-top, etc.

Solo abajo 
<h2 style="border-bottom: 3px solid orange;">
  Título Subrayado
</h2>
