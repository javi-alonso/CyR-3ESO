4. El Modelo de Caja (Box Model)

Todo es una caja

En HTML y CSS, cada elemento es una caja rectangular. Entender esto es VITAL. La caja tiene 4 partes:

  1. Content (Contenido): El texto o imagen real.
  2. Padding (Relleno): Espacio entre el contenido y el borde. (Es transparente).
  3. Border (Borde): La línea que rodea el relleno.
  4. Margin (Margen): Espacio fuera del borde, para separar la caja de otras cajas.

Ejemplo Visual

Ver las partes 
<style>
  .caja-ejemplo {
    width: 200px;         /* Ancho del contenido */
    padding: 20px;        /* Espacio interno (verde claro) */
    border: 5px solid black; /* Borde */
    margin: 30px;         /* Espacio externo (naranja) */
    background-color: lightblue; /* Color del contenido */
  }
</style>

<div class="caja-ejemplo">
  Soy el contenido.
</div>
Soy el contenido.
(El área naranja es el margen)

Padding vs Margin

La diferencia clave: El Padding es parte de la caja (tiene el color de fondo). El Margin es espacio vacío fuera de la caja.

Comparación 
<style>
  .con-padding {
    background: pink;
    padding: 20px; /* Crece la caja */
  }
  .con-margin {
    background: pink;
    margin: 20px; /* Se aleja de los lados */
  }
</style>

<button class="con-padding">Botón Grande (Padding)</button>
<button class="con-margin">Botón Separado (Margin)</button>
