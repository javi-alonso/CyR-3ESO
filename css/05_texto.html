5. Estilos de Texto

Tipografía Básica

El texto es lo más importante de la web. CSS nos da control total sobre cómo se ve.

Alineación y Decoración

Formato de Texto 
<style>
  .centro { text-align: center; }
  .derecha { text-align: right; }
  .tachado { text-decoration: line-through; }
  .subrayado { text-decoration: underline; }
  .mayus { text-transform: uppercase; }
</style>

<p class="centro">Estoy centrado</p>
<p class="derecha">Estoy a la derecha</p>
<p class="tachado">No me leas</p>
<p class="mayus">Gritando en mayúsculas</p>

Espaciado y Altura de Línea

Para hacer el texto legible, ajustamos el espacio entre líneas (line-height) y letras (letter-spacing).

Texto legible 
<style>
  .texto-denso {
    line-height: 1;
    letter-spacing: -1px;
  }
  .texto-aireado {
    line-height: 1.8;
    letter-spacing: 2px;
  }
</style>

<p class="texto-denso">Texto muy pegado difícil de leer.</p>
<p class="texto-aireado">Texto con espacio, más elegante y fácil de leer.</p>
