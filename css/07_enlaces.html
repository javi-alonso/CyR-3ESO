7. Estilo en Enlaces

Personalizando la navegación

Por defecto, los enlaces son azules y subrayados. CSS nos permite transformarlos en botones o textos elegantes.

El estado :hover

Podemos cambiar el estilo cuando el ratón pasa por encima usando :hover.

Ejemplo 1: Enlace Texto Moderno

Enlace sin subrayado 
<style>
  .enlace-cool {
    color: #333;
    text-decoration: none; /* Quita el subrayado */
    font-weight: bold;
    border-bottom: 2px solid transparent;
    transition: 0.3s; /* Animación suave */
  }
  .enlace-cool:hover {
    color: #2f81f7;
    border-bottom: 2px solid #2f81f7;
  }
</style>

<a href="#" class="enlace-cool">Pasa el ratón por aquí</a>
Pasa el ratón por aquí

Ejemplo 2: Creando un Botón

Los botones de esta web son en realidad enlaces <a> estilizados.

Botón CSS 
<style>
  .boton-link {
    background-color: #4CAF50;
    color: white;
    padding: 10px 20px;
    text-decoration: none;
    border-radius: 5px;
    display: inline-block;
  }
  .boton-link:hover {
    background-color: #45a049;
    transform: translateY(-2px); /* Se mueve un poco hacia arriba */
  }
</style>

<a href="#" class="boton-link">¡Soy un botón!</a>
