8. Estilo en Listas

Personalizando Viñetas

Las listas no tienen por qué ser aburridos puntos negros.

Cambiando el marcador

Usamos list-style-type para cambiar la forma del punto o número.

Tipos de Lista 
<style>
  .cuadrado { list-style-type: square; }
  .circulo { list-style-type: circle; }
  .romanos { list-style-type: upper-roman; }
  .nada { list-style-type: none; padding: 0; }
</style>

<ul class="cuadrado"><li>Cuadrado</li></ul>
<ul class="circulo"><li>Círculo</li></ul>
<ol class="romanos"><li>I, II, III...</li></ol>
  • Cuadrado
  • Círculo
  1. I, II, III...

Listas como Menús

Al quitar los puntos (list-style: none), las listas son perfectas para menús de navegación.

Menú Horizontal 
<style>
  .menu {
    list-style: none; /* Adiós puntos */
    padding: 0;
    display: flex;    /* Se ponen en fila */
    gap: 15px;        /* Espacio entre ellos */
    background: #333;
    padding: 10px;
  }
  .menu li { color: white; }
</style>

<ul class="menu">
  <li>Inicio</li>
  <li>Productos</li>
  <li>Contacto</li>
</ul>
  • Inicio
  • Productos
  • Contacto
