9. Estilo en Tablas

Tablas Legibles y Bonitas

Las tablas por defecto son feas. Con CSS podemos hacerlas muy profesionales.

Bordes Colapsados

IMPORTANTE: Para que los bordes se vean finos y unidos, usamos border-collapse: collapse;.

Tabla Básica Estilizada 
<style>
  table {
    width: 100%;
    border-collapse: collapse; /* Une los bordes dobles */
  }
  th, td {
    border: 1px solid #ddd;
    padding: 8px; /* Espacio para respirar */
    text-align: left;
  }
  th {
    background-color: #f2f2f2;
  }
</style>

<table>
  <tr><th>Nombre</th><th>Puntos</th></tr>
  <tr><td>Mario</td><td>100</td></tr>
  <tr><td>Luigi</td><td>90</td></tr>
</table>
Nombre Puntos
Mario 100
Luigi 90

Efecto Cebra (Zebra Striping)

Para facilitar la lectura, pintamos las filas pares de otro color usando :nth-child(even).

tr:nth-child(even) {
  background-color: #f9f9f9;
}
