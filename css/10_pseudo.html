10. Pseudoclases

Estados especiales

Las pseudoclases permiten cambiar el estilo de un elemento bajo ciertas condiciones (ej. cuando pasas el ratón). Empiezan con dos puntos :.

El rey: :hover

Cambia el estilo al pasar el cursor por encima.

Interactividad 
<style>
  .caja-interactiva {
    width: 100px;
    height: 100px;
    background: blue;
    transition: 0.5s; /* Animación suave */
  }
  .caja-interactiva:hover {
    background: red;
    width: 200px; /* Crece al pasar el ratón */
  }
</style>

<div class="caja-interactiva"></div>

Otras Pseudoclases útiles

  • :active: Mientras haces clic (como pulsar un botón físico).
  • :focus: Cuando estás escribiendo en un campo de texto.
  • :first-child: Selecciona solo el primer elemento de un grupo.
Input Focus 
<style>
  input { border: 1px solid #ccc; padding: 5px; }
  input:focus {
    background-color: #e0f7fa;
    border-color: #00bcd4;
    outline: none; /* Quita el borde azul feo del navegador */
  }
</style>

<input type="text" placeholder="Haz clic aquí...">
