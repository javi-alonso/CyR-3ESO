11. Unidades de Medida

¿Cuánto mide esto?

En CSS no usamos centímetros. Usamos unidades para pantallas.

Tipos Principales

  • px (Píxeles): Unidad fija. Un punto en la pantalla. Fácil de entender pero no escala bien.
  • % (Porcentaje): Relativo al padre. Si el padre mide 200px, 50% son 100px. Ideal para diseños elásticos.
  • rem: Relativo al tamaño de letra raíz (normalmente 16px). 1rem = 16px. Es la mejor para accesibilidad.
  • vh / vw: Porcentaje del tamaño de la ventana (Viewport Height/Width).

Ejemplo: px vs %

Relativo vs Absoluto 
<div style="width: 300px; border: 2px solid black; padding: 10px;">
  Soy el padre (300px de ancho)
  
  <div style="width: 50%; background: orange;">
    Soy el 50% (Mido 150px)
  </div>
  
  <div style="width: 50px; background: blue; color: white;">
    Soy 50px fijos
  </div>
</div>
Soy el padre (ancho limitado)
50%
50px

Unidades de Pantalla (Visual)

width: 100vw; ocupa TODO el ancho de la pantalla, sin importar márgenes.

height: 100vh; ocupa TODA la altura visible.
