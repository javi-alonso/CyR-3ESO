Curso de CSS

Cascading Style Sheets: El arte de diseñar la web.

Introducción y Sintaxis

CSS se usa para dar estilo a los documentos HTML. La sintaxis básica consiste en un selector y un bloque de declaración (propiedad: valor).

Sintaxis Básica 
<style>
  p {
    color: blue;
    font-size: 18px;
  }
</style>

<p>Este texto será azul y grande.</p>

Este texto será azul y grande.

Fondos (Background)

Podemos cambiar el color de fondo o usar una imagen.

Color e Imagen de Fondo 
<style>
  .fondo-color {
    background-color: #f0f0f0;
    padding: 10px;
  }
  .fondo-imagen {
    background-image: url('https://placehold.co/600x400');
    background-repeat: no-repeat;
    background-size: cover;
    color: white;
    padding: 20px;
  }
</style>

<div class="fondo-color">Fondo gris claro</div>
<div class="fondo-imagen">Texto sobre imagen</div>
Fondo gris claro
Texto sobre imagen

Bordes

La propiedad border define el ancho, estilo y color del borde.

Bordes y Redondeado 
<style>
  .caja-borde {
    border: 2px solid red;
    border-radius: 10px; /* Bordes redondeados */
    padding: 10px;
  }
</style>

<div class="caja-borde">Tengo un borde rojo y redondeado.</div>
Tengo un borde rojo y redondeado.

Ancho (Width) y Alto (Height)

Controlamos las dimensiones de los elementos con width y height.

Dimensiones 
<style>
  .caja-dimension {
    width: 50%;
    height: 50px;
    background-color: coral;
  }
</style>

<div class="caja-dimension">Ocupo el 50% del ancho</div>
Ocupo el 50% del ancho

Estilos de Texto

Propiedades como text-align, text-decoration, text-transform y line-height dan forma al texto.

Formato de Texto 
<style>
  .centro { text-align: center; }
  .tachado { text-decoration: line-through; }
  .mayus { text-transform: uppercase; }
</style>

<p class="centro">Texto centrado</p>
<p class="tachado">Texto tachado</p>
<p class="mayus">Texto en mayúsculas</p>

Texto centrado

Texto tachado

Texto en mayúsculas

Fuentes (Google Fonts)

Para usar una fuente de Google Fonts, primero la importamos en el <head> y luego la usamos en font-family.

Google Fonts 
<!-- En el head -->
<link href="https://fonts.googleapis.com/css2?family=Pacifico&display=swap" rel="stylesheet">

<style>
  .fuente-chula {
    font-family: 'Pacifico', cursive;
    font-size: 24px;
  }
</style>

<p class="fuente-chula">¡Hola con estilo!</p>

¡Hola con estilo!

Estilo en Enlaces

Podemos cambiar el color y quitar el subrayado a los enlaces.

Botones con Enlaces 
<style>
  .boton-link {
    text-decoration: none;
    background-color: #4CAF50;
    color: white;
    padding: 10px 20px;
    border-radius: 5px;
    display: inline-block;
  }
  .boton-link:hover {
    background-color: #45a049;
  }
</style>

<a href="#" class="boton-link">Soy un botón</a>
Soy un botón

Estilos de Listas

Podemos cambiar las viñetas con list-style-type.

Listas Personalizadas 
<style>
  ul.cuadrados { list-style-type: square; }
  ul.sin-puntos { list-style-type: none; padding: 0; }
</style>

<ul class="cuadrados">
  <li>Item Cuadrado 1</li>
  <li>Item Cuadrado 2</li>
</ul>
  • Item Cuadrado 1
  • Item Cuadrado 2

Estilos de Tablas

Las tablas bonitas usan border-collapse y padding.

Tabla Estilizada 
<style>
  table.bonita {
    width: 100%;
    border-collapse: collapse;
  }
  table.bonita th, table.bonita td {
    padding: 10px;
    border-bottom: 1px solid #ddd;
    text-align: left;
  }
  table.bonita th {
    background-color: #f2f2f2;
    color: #333;
  }
</style>

<table class="bonita">
  <tr><th>Producto</th><th>Precio</th></tr>
  <tr><td>Manzanas</td><td>1.50€</td></tr>
  <tr><td>Naranjas</td><td>2.00€</td></tr>
</table>
Producto Precio
Manzanas 1.50€
Naranjas 2.00€

Pseudoclases

Estados especiales de los elementos, como cuando pasamos el ratón por encima (:hover).

Efecto Hover 
<style>
  .hover-box {
    width: 100px;
    height: 100px;
    background-color: blue;
    transition: background-color 0.3s;
  }
  .hover-box:hover {
    background-color: red;
  }
</style>

<div class="hover-box"></div>

Px vs %

</div>
