4. Enlaces (Links)

Conectando la Web

La "H" de HTML significa HyperText (Hipertexto). Esto significa texto con enlaces. La etiqueta mágica es <a> (de "anchor" o ancla).

El atributo href

Es obligatorio. Indica "hacia dónde" va el enlace.

Enlace Básico 
<p>
  Visita el buscador: 
  <a href="https://www.google.com">Ir a Google</a>
</p>

Visita el buscador: Ir a Google

Abrir en nueva pestaña

Usamos target="_blank" para que el usuario no pierda tu página al hacer clic.

Nueva Pestaña 
<a href="https://www.wikipedia.org" target="_blank">
  Abrir Wikipedia en otra pestaña
</a>
Abrir Wikipedia en otra pestaña
