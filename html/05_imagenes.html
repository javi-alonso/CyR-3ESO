5. Imágenes

Una imagen vale más que mil palabras

Para insertar imágenes usamos <img>. Es una etiqueta "vacía", no tiene etiqueta de cierre.

Atributos imprescindibles

  • src: "Source" (Fuente). La URL o ruta del archivo de imagen.
  • alt: "Alternative text". Texto que describe la imagen si no carga o para ciegos.

Ejemplo Básico

Insertar Imagen 
<img 
  src="https://via.placeholder.com/300x150" 
  alt="Un rectángulo gris de ejemplo"
>

Controlando el Tamaño

Aunque es mejor hacerlo con CSS, HTML permite definir width (ancho) y height (alto) en píxeles.

Imagen Pequeña 
<!-- Forzamos la imagen a 100px de ancho -->
<img 
  src="https://via.placeholder.com/300x150" 
  alt="Miniatura"
  width="100"
>
⚠️ Error Común

No olvides el atributo alt. Es vital para la accesibilidad y Google te penaliza si no lo usas.
