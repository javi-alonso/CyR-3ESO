6. Listas

Ordenando ideas

Las listas son fundamentales para menús, índices o simples enumeraciones. Hay dos tipos principales:

Tipos de Listas

  • <ul> (Unordered List): Lista con viñetas (puntos). El orden no importa.
  • <ol> (Ordered List): Lista numerada (1, 2, 3...). El orden importa.
  • <li> (List Item): Cada elemento de la lista.

Ejemplo: Lista de la Compra (Desordenada)

Lista con puntos 
<h3>Ingredientes</h3>
<ul>
  <li>Harina</li>
  <li>Huevos</li>
  <li>Leche</li>
</ul>

Ingredientes

  • Harina
  • Huevos
  • Leche

Ejemplo: Top 3 Canciones (Ordenada)

Lista Numerada 
<h3>Ranking</h3>
<ol>
  <li>Bohemian Rhapsody</li>
  <li>Imagine</li>
  <li>Smells Like Teen Spirit</li>
</ol>

Ranking

  1. Bohemian Rhapsody
  2. Imagine
  3. Smells Like Teen Spirit

Listas Anidadas

Puedes poner una lista dentro de otra.

Anidación 
<ul>
  <li>Frutas
    <ul>
      <li>Manzana</li>
      <li>Pera</li>
    </ul>
  </li>
  <li>Verduras</li>
</ul>
  • Frutas
    • Manzana
    • Pera
  • Verduras
