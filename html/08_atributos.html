8. Atributos, ID y Clases

Dando superpoderes a las etiquetas

Los atributos ofrecen información extra sobre las etiquetas. Algunos son específicos (como src en imágenes) pero otros son globales (valen para todos).

ID vs Class

Estos son los dos atributos más usados para conectar HTML con CSS y JavaScript.

  • id: Identificador ÚNICO. Solo puede haber uno con ese nombre en toda la página. Como el DNI.
  • class: Identificador múltiple. Varios elementos pueden compartir la misma clase. Como el uniforme del colegio.

Ejemplo: ID y Class

Identificadores y Clases 
<!-- Solo hay un encabezado principal -->
<h1 id="titulo-principal">Mi Web</h1>

<!-- Varios párrafos destacados -->
<p class="destacado">Noticia urgente 1.</p>
<p class="destacado">Noticia urgente 2.</p>

<p>Noticia normal.</p>

Mi Web

Noticia urgente 1.

Noticia urgente 2.

Noticia normal.

El atributo Style

Permite escribir CSS directamente dentro de la etiqueta. (Nota: Es mejor usar un archivo CSS aparte, pero es bueno conocerlo).

Estilos en Línea 
<p style="color: red; font-size: 20px;">
  Soy rojo y grande
</p>

Soy rojo y grande
