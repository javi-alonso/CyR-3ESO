Curso de HTML

El lenguaje de marcado que estructura la web.

Estructura Básica

Todo documento HTML comienza con una estructura esqueleto estándar. Aquí es donde definimos el tipo de documento y las secciones principales: <head> (información técnica) y <body> (contenido visible).

Ejemplo de Estructura 
<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Mi Primera Página</title>
  </head>
  <body>
    <h1>¡Hola Mundo!</h1>
    <p>Este es mi primer párrafo.</p>
  </body>
</html>

¡Hola Mundo!

Este es mi primer párrafo.

Encabezados y Párrafos

HTML nos ofrece 6 niveles de encabezados, desde <h1> (el más importante) hasta <h6>. Los párrafos se definen con <p>.

Ejemplo de Texto 
<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<p>Esto es un párrafo de texto normal.</p>
<p>Podemos usar <strong>negrita</strong> y <em>cursiva</em>.</p>

Título Principal

Subtítulo

Esto es un párrafo de texto normal.

Podemos usar negrita y cursiva.

Semántica

Las etiquetas semánticas dan significado al contenido, ayudando a los navegadores y buscadores a entender la estructura. Ejemplos: <header>, <nav>, <main>, <footer>, <article>.

Ejemplo Semántico 
<header>
  <nav>Menú de navegación</nav>
</header>
<main>
  <article>
    <h2>Título del Artículo</h2>
    <p>Contenido del artículo...</p>
  </article>
</main>
<footer>Pie de página</footer>
Header

Título del Artículo

Contenido del artículo...

Pie de página

Enlaces

El atributo principal es href, que indica el destino del enlace. Se usa la etiqueta <a>.

Ejemplo de Enlace 
<a href="https://www.google.com" target="_blank">Ir a Google</a>
<br>
<a href="index.html">Volver al Inicio</a>
Ir a Google
Volver al Inicio

Imágenes

Usamos la etiqueta <img>. Requiere el atributo src (fuente) y alt (texto alternativo).

Ejemplo de Imagen 
<img src="https://via.placeholder.com/150" alt="Imagen de ejemplo">

Listas

Hay dos tipos principales: Ordenadas (<ol>) y Desordenadas (<ul>). Cada elemento es un <li>.

Ejemplo de Listas 
<h3>Lista Desordenada</h3>
<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>

<h3>Lista Ordenada</h3>
<ol>
  <li>Primer paso</li>
  <li>Segundo paso</li>
</ol>

Lista Desordenada

  • HTML
  • CSS

  2. Segundo paso

Tablas

Las tablas se crean con <table>. Las filas son <tr>, los encabezados <th> y las celdas de datos <td>.

Ejemplo de Tabla 
<table border="1">
  <thead>
    <tr>
      <th>Nombre</th>
      <th>Edad</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Ana</td>
      <td>15</td>
    </tr>
    <tr>
      <td>Luis</td>
      <td>16</td>
    </tr>
  </tbody>
</table>
Nombre Edad
Ana 15
Luis 16

Atributos, ID y Clases

Los atributos modifican las etiquetas. id es un identificador único, y class sirve para agrupar elementos (muy usado en CSS).

Ejemplo de Atributos 
<!-- ID único -->
<p id="intro">Este párrafo tiene un ID único.</p>

<!-- Clases compartidas -->
<p class="destacado">Texto destacado 1</p>
<p class="destacado">Texto destacado 2</p>

<!-- Atributo style (CSS en línea) -->
<p style="color: red;">Texto en rojo</p>

Este párrafo tiene un ID único.

Texto destacado 1 (simulado)

Texto destacado 2 (simulado)
